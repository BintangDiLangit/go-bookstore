# Go Bookstore

Build with Go 1.19.3

## Installation

Run Locally

```bash
  git clone https://github.com/BintangDiLangit/go-bookstore.git
  cd go-bookstore
  go build
  go main.go
```

Open at browser (firefox, google chrome, etc)

```bash
  http://127.0.0.1:9010
```
